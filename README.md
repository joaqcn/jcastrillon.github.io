# Joaquin Castrillon Portfolio

A full-screen interactive terminal portfolio built with React and Vite.

## Available commands

- `start`
- `about`
- `education`
- `experience`
- `skills`
- `projects`
- `contact`
- `help`
- `clear`

## Local development

```bash
npm install
npm run dev
```

## Production build

```bash
npm run build
```
